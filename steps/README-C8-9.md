# 用 hook 完成提取共用的 api

- 取学生、课程及教师基本信息的 api，大部分查询条件和返回结果形式都相同，用 hook 完成，参数包含 api 名称，api 参数，是否保存原有查询结果，返回数据类型。
- 用 useEffect 完成 api 的调用，将当前页面，设置换页，当前数据，总数据等数据和函数返回
